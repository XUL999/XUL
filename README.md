## 🌟 项目简介

XUL 是全球首个基于 AI 原生理念设计的 Layer-1 公链，主张在区块链不可篡改的基础上，构建**可验证、可审计、可持续演化**的 AI 服务基础设施。

## 🎯 核心理念

### 三条核心法则

| 法则 | 描述 |
|------|------|
| **主权法则** | 数据与模型主权归用户所有 |
| **可验法则** | AI 的每一次决策，链上可追溯、可审计 |
| **演化法则** | 持续演化、适应与创造 |
